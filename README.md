# What A Team — Resources

Public website for **What A Team**, a free mobile app for small teams to celebrate their values together.

This repository hosts the app's website (via GitHub Pages), including:

- Landing page
- Privacy policy
- Support / FAQ
- Impressum
- Issue templates for bug reports and feature requests

## App

What A Team is a peer recognition app for small teams (2-20 people). Team members co-create their values and recognize each other through "High Fives" tied to those values.

- Free, no subscriptions, no ads
- iOS and Android
- Built with Flutter and Firebase
